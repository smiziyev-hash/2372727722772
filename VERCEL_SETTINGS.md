# Настройки Vercel для Qwik Edge

## Настройки проекта на Vercel

Откройте проект на Vercel → Settings → General:

1. **Root Directory**: `web`
2. **Build Command**: `npm run build.server`
3. **Output Directory**: (оставить пустым)
4. **Install Command**: `npm install --legacy-peer-deps`
5. **Framework Preset**: `Other`

## Важно

После изменения настроек нажмите **"Redeploy"** в разделе General для применения изменений.
