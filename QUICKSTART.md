# Быстрый старт InkSec

## Запуск проекта

1. Перейдите в директорию проекта:
```bash
cd web
```

2. Установите зависимости (если еще не установлены):
```bash
npm install --legacy-peer-deps
```

3. Запустите сервер разработки:
```bash
npm run dev
```

4. Откройте браузер и перейдите по адресу:
```
http://localhost:5173
```

## Переключение языков

В правом верхнем углу навигационного меню есть выпадающий список для переключения между:
- English (EN)
- Русский (RU)

## Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен сервер.

## Дополнительные команды

- `npm run build` - сборка для продакшена
- `npm run preview` - предпросмотр продакшен сборки
- `npm run lint` - проверка кода линтером

## Примечания

- Проект использует Qwik framework
- Все переводы находятся в `web/src/i18n/translations.ts`
- Настройки проекта в `web/package.json`
- Статические файлы в `web/public/`
